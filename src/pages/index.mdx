## Sessions

<div class="grid grid-cols-1 sm:grid-cols-2 prose-p:my-0">

Start new session

- <T>tmux</T>
- <T>tmux new</T>
- <T>tmux new-session</T>

Rename session

- <K>$</K>

Start a new session by name

- <T>tmux new-session -s NAME</T>
- <M client:load>new -s NAME</M>

Start or attach to session by name

- <T>tmux new-session -A -s NAME</T>

Kill current session

- <M>kill-session</M>

Kill session by name

- <T>tmux kill-session -t NAME</T>
- <T>tmux kill-ses -t NAME</T>

Kill other sessions

- <T>tmux kill-session -a</T>

Kill all sessions except NAME

- <T>tmux kill-session -a -t NAME</T>

Rename session

- <K>$</K>

Detach from session

- <K>d</K>

Detach others on the session (todo)

- <M>attach -d</M>

Show all sessions

- <T>tmux ls</T>
- <T>tmux list-sessions</T>
- <K>s</K>

Show all sessions w/ windows

- <K>w</K>

Attach to last session

- <T>tmux a</T>
- <T>tmux at</T>
- <T>tmux attach</T>
- <T>tmux attach-session</T>

Attach to last session by NAME

- <T>tmux a -t NAME</T>
- <T>tmux at -t NAME</T>
- <T>tmux attach -t NAME</T>
- <T>tmux attach-session -t NAME</T>

Goto previous session

- <K>(</K>

Goto next session

- <K>)</K>

## Windows

<div class="grid grid-cols-1 sm:grid-cols-2 prose-p:my-0">

Start session with session and window name

- <T>tmux new -s SESSION_NAME -n WINDOW_NAME</T>

Create window

- <K>c</K>

Rename current window

- <K>,</K>

Close current window

- <K>&</K>

List windows

- <K>w</K>

Goto previous window

- <K>p</K>

Goto next window

- <K>n</K>

Switch window by number

- <K>0 ... 9</K>

Switch to last active window

- <K>l</K>

<div>

Swap window order (absolute)

- `-s` = source
- `-t` = target

</div>

- <M>swap-window -s 2 -t</M>

Swap window order (relative)

- <M>swap-window -t -1</M>

Move window between sessions (by name)

- <M>movew -s foo:0 -t bar:9</M>

Move window between sessions (by number)

- <M>movew -s 0:0 -t 1:9</M>

Reposition window in the current Session (todo)

- <M>move-window -s src_session:src_window</M>
- <M>movew -s 0:9</M>

Renumber windows to remove gap in the sequence (todo)

- <M>move-window -r</M>
- <M>movew -r</M>
